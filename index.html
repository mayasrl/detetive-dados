A Grande Conspiração dos Dados

No ano 2091, a cidade de Neon-City era governada por um sistema perfeito de dados interconectados. Cada cidadão, cada transação, cada movimento era registrado no Grande Banco de Dados Central.

Mas algo terrível aconteceu... Os dados começaram a desaparecer, se corromper e se desorganizar. A Agência de Harmonia Digital descobriu sinais de uma conspiração interna.

Você foi recrutado como detetive especialista em dados para desvendar este mistério e restaurar a ordem digital antes que o caos tome conta da cidade.
